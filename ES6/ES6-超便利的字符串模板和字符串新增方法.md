- **`` 字符串模板**
```
特点：可以随意的换行，中间变量使用${xxx}，大括号里面能识别js代码；
如：
let name = 'strive';
let age = '22';
let str = `我叫
${name},
我今年
${age}
岁了`；
```
- **字符串新增**
```
1.新增字符串查找方法
（1）includes(要找的东西)
  字符串是否包含要找的东西，返回 true/false
  应用：检查用户使用的浏览器
（2）startsWith(检测的东西)
  字符串是否已谁开头，返回 true/false
  应用：检查路径
（3）endsWith(检测的东西)
  是否已什么结尾，返回 true/false
  应用：检查文件格式

2.字符串重复
repeat(次数)
参数表示字符串重复次数
返回一个重复包含n个初始字符串的新字符串

3.填充字符串
（1）padStart(整个字符串的长度，填充的东西) 
  往字符串前面填充插入字符串

（2）padEnd(整个字符串的长度，填充的东西)  
  往字符串后面填充插入字符串
```
