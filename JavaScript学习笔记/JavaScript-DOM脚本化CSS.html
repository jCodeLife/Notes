JavaScript DOM脚本化CSS

可以通过DOM元素来控制其CSS样式，即可读写其中的样式

读写CSS属性

格式

dom.style.prop

规则：

1.通过dom.style.prop可以读写行间样式，没有兼容性问题，碰到float这样的系统保留字属性，前面应加css；如float-->cssFloat;

2.符合属性建议拆解；

3.组合单词变成小驼峰式；

4.写入的值必须是字符串格式；

*其实，style属性普遍存在于dom元素中，是个dom元素就有style属性，其对应的是CSSStyleDeclaration对象，这个样式声明对象能将这个dom元素所有能用的属性都列出来，巧了，这刚好能改变CSS样式。

查询计算样式

通过计算样式，可以获得当前元素所展示的一切CSS的显示值

1.window.getComputedStyle(ele,null)

(1)计算样式只读；

(2)返回的计算样式的值都是绝对值，没有像em、n%的相对单位

(3)IE8及以下不兼容；

2.ele.currentStyle()

(1)计算样式只读；

(2)返回的计算样式的值不是经过转换的绝对值；

(3)IE独有的
